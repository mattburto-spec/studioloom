# 🎨 Questerra — MYP Design Adventure

A gamified learning platform for MYP Design students to work through the design cycle.

## Quick Start (Local Development)

```bash
npm install
npm run dev
```

Then open http://localhost:5173/questerra/

## Deploy to GitHub Pages

1. Create a new repo called `questerra` on GitHub
2. Push this code:

```bash
git init
git add .
git commit -m "initial questerra mockup"
git remote add origin https://github.com/YOUR-USERNAME/questerra.git
git branch -M main
git push -u origin main
```

3. Go to repo → **Settings** → **Pages** → Source: **GitHub Actions**
4. The workflow will run automatically and your site will be live at:
   `https://YOUR-USERNAME.github.io/questerra/`

## Demo Login

- **Student**: Enter any class code + name
- **Teacher**: Enter any email + password
