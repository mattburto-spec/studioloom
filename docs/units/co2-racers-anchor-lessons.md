## CLASS 1 (~60 min) — Unit launch

**Phase mapping:** opening → miniLesson → workTime → debrief

| # | Block | Phase | Where to find it | What you fill in |
|---|---|---|---|---|
| 1 | **Teacher Notes** "Hook video 1" | opening | Content category | Embed clip A (kid showing agency in a tough scenario). Caption: *"Watch this. Don't say anything yet."* |
| 2 | **Teacher Notes** "Hook video 2" | opening | Content category | Embed clip B (different kid, same scenario, no agency). Caption: *"Now watch this. Which kid is showing agency? Why?"* |
| 3 | **Written Response** "Co-construct definition" | opening | Response category | Prompt: *"In your own words: what does AGENCY mean for the next 14 weeks of design? Pair-share with the person next to you, then write."* |
| 4 | **Key Concepts** "Three Cs intro" | miniLesson | Content category | List: **Choice** (autonomy), **Causation** (because-clauses), **Change** (iteration). One sentence per C. |
| 5 | **Strategy Canvas** ⭐ | workTime | Response category | Pre-built — 3 prompts already configured (Design philosophy / Biggest risk / Fallback plan). Set duration 10m. |
| 6 | **Teacher Notes** "Now: backward-map" | workTime | Content category | Brief: *"Open your Project Board (right rail → Timeline). Set race day = June 11. Add 4 milestones working backward: Final design → CAD complete → Bandsaw cuts done → First sketches. Drag dot positions = your time estimates."* |
| 7 | **Teacher Notes** "Now: set up Kanban" | workTime | Content category | Brief: *"Project Board → Kanban (below Timeline). Add 5 starter cards in Backlog: Concept sketches × 3, Working drawing, CAD wheel. We'll add more as the unit progresses."* |
| 8 | **New Metrics** "Survey 1: Baseline" ⭐ | debrief | New Metrics category (1 block per element you're tracking) | This is your baseline. **Click each element you want students rating** (typically all 3 Three Cs anchors). Students rate themselves end-of-class. |
| 9 | **Process Journal** "Closing" | debrief | Response category | Pre-built — Did/Noticed/Decided/Next. Set duration 5m. *"What does agency mean for YOU in this unit, given today?"* |

**⭐ The two anchors that didn't exist before today: Strategy Canvas + the NM survey blocks. The rest is standard editor blocks.**

**Total time:** Hook 10m + Definition 5m + Three Cs intro 5m + Strategy Canvas 10m + Backward-map 10m + Kanban setup 10m + Survey 5m + Closing journal 5m = ~60m.

## CLASS 7 (~60 min) — Mid-unit critique

**Phase mapping:** opening → miniLesson → workTime → debrief

| # | Block | Phase | Where to find it | What you fill in |
|---|---|---|---|---|
| 1 | **Teacher Notes** "Today's frame" | opening | Content category | *"Mid-unit. You've made decisions. Today we critique each other's decisions, not each other's wheels. Read your journals first."* |
| 2 | **Self-Reread** ⭐ | miniLesson | Response category | Pre-built — single deep prompt. Set duration 10m. |
| 3 | **Teacher Notes** "Pair up" | workTime | Content category | *"Random pairs. 4 pairs + 1 trio for class of 9. Pair list on the board."* (You handle this in-class.) |
| 4 | **Teacher Notes** "Decision share" | workTime | Content category | *"Each person shares ONE decision they made — drawn from your Decision Log OR a Kanban card's because-clause. 5 min each."* |
| 5 | **Key Concepts** "Kind / Specific / Helpful" | workTime | Content category | Three short bullets:<br>**Kind** — assume good intent.<br>**Specific** — point to the exact decision, not the whole project.<br>**Helpful** — ask a question or offer a concrete suggestion. |
| 6 | **Written Response** "Feedback exchange" | workTime | Response category | Prompt: *"Receive feedback (10 min), then give feedback (10 min). Write down the most useful piece you received."* |
| 7 | **Written Response** "What will I act on?" | workTime | Response category | Prompt: *"What's ONE piece of feedback you'll act on? What new Kanban card does this become?"* (After they save, the new card appears in their Project Board because the journal block triggers Kanban auto-create — but Written Response doesn't, so they'll need to add it manually OR you can swap this for a Process Journal block.) |
| 8 | **Strategy Canvas** ⭐ "Re-prompt" | debrief | Response category | **Same pre-built block.** Add a new instance to this lesson — it'll create a fresh entry. Duration 5m. Header text override: *"Mid-unit re-read. Anything changed about your philosophy / risk / fallback?"* |
| 9 | **New Metrics** "Survey 3: Mid-unit" | debrief | New Metrics category | Same Three Cs elements as Survey 1. |

**⭐ Self-Reread + the second Strategy Canvas instance are the new content this round.**

## Calibration triggers (Surveys 2 + 4)

These don't get their own anchor lessons — they fire when a student
ticks a milestone on their personal Timeline (Working drawing complete →
Survey 2; Painting complete → Survey 4).

**Per the spec, the milestone-triggered surveys are NOT YET WIRED.** When
a student completes a milestone today, no survey auto-prompts. Two
options:

- **Option A (recommended for v1 pilot):** announce the trigger
  manually — "now that you're done with your working drawing, do
  Survey 2 in the lesson". Add an NM block to whichever lesson
  typically falls right after the working-drawing milestone.
- **Option B (deferred):** wire timeline milestone-done event →
  Survey trigger. Filed as **FU-AGENCY-MILESTONE-SURVEY-AUTOTRIGGER**
  (LOW). Probably worth doing post-pilot once you've felt the manual
  flow.

## Smoke order (for AG.6)

When you're ready to verify:

1. **Teacher side:** Author the 3 anchor lessons via this guide. Save.
   Open the Class Hub → Metrics tab → confirm tracking is on +
   elements selected. Open Class Hub → Attention tab → confirm empty
   panel renders + 🛑 banner shows.
2. **Student side (test student):** Run through Class 1 end-to-end.
   Verify Strategy Canvas saves to Portfolio + appears in Narrative.
   Verify Survey 1 entries hit the Three Cs aggregator. Verify the
   closing Process Journal creates a Kanban backlog card from Next.
3. **Re-load Class 1 as that student.** Strategy Canvas + Journal
   should show as saved-preview blocks (round 11 fix). Click Edit —
   prompts pre-fill (round 4 fix).
4. **Open Portfolio → Narrative.** All saved entries should appear,
   including any photos you uploaded (round 10 fix).
5. **Teacher side again:** Open Attention panel. The student should
   appear with `Journal: today`, `Kanban: today`, `Calibration: never`,
   Three Cs aggregate based on Survey 1 numbers. Click their row —
   Calibration mini-view opens with side-by-side ratings (round 8/9).
   Save an observation; panel reloads with `Calibration: today`.

If all 5 pass, AG.1–AG.6 is smoke-verified for one student × one
lesson. Repeat with a second test student to spot per-student state
issues.
